To Do List 

Aplicación web de lista de tareas desarrollada con HTML, CSS y JavaScript puro, enfocada en ofrecer una experiencia clara.

---

Demo

https://starkiller6637.github.io/Todolist/

Funcionalidades

* Agregar tareas
* Editar tareas existentes
* Marcar tareas como completadas
* Eliminar tareas

* Filtrar tareas por estado:

  * Todas
  * Pendientes
  * Completadas

* Filtrar por sección:

  * Todas		
  * Escuela
  * Trabajo
  * Casa
  * Personal 

* Vista por tipo:

  * Todo
  * Hoy
  * Próximos días
  * Importantes

* Asignar fecha a cada tarea
* Marcar tareas como importantes
* Contador de tareas pendientes
* Limpiar tareas completadas
* Persistencia de datos mediante localStorage
* Modo claro y modo oscuro
* Diseño adaptable a diferentes tamaños de pantalla

---

Tecnologías utilizadas

* HTML5

* JavaScript (Vanilla JS)
* LocalStorage

Decisiones de diseño

* Se utilizó JavaScript puro para demostrar manejo del DOM sin frameworks
* Se implementó localStorage para mantener la información entre sesiones
* La interfaz se diseñó priorizando claridad, orden visual y facilidad de uso
* La estructura del código busca ser simple, entendible y mantenible
